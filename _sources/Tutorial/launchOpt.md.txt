# 起動オプション

##PTSIMの起動オプション

通常、シミュレーション体系を確認するためにGUI環境でインタラクティブモードでの可視化を行います。
そして、確認ができたら照射数(/run/beamOn)を増やして、可視化をオフにしたバッチモードを用います。

PTSIMの起動オプションには、以下の表のものがあります。
| オプション | 説明 |
| ---:| :--- |
| -t {n} | {n}:スレッド数, デフォルト=2 |
| -u {utype} | {utype}:ユーザインターフェイス tcsh またはqt, デフォルト=qt |
| -i {macro}| {macro}:マクロファイル。インタラクティブモード。<br>
起動後にマクロファイルのコマンドを実行したのちに、コマンド入力待ち状態となる。|
| -m {macro}| {macro}:マクロファイル。バッチモード。<br>
起動してマクロファイルのコマンドを実行して終了する。|

### インタラクティブモードでの起動例

#### GUI環境での確認や試行作業
```
$ ./bin/PTSdemo  -u qt  -i  Sample.mac
```
QQウインドウが起動し、コマンド入力待ち状態になる。

#### CUI環境での確認や試行作業
```
$ ./bin/PTSdemo  -u tcsh  -i  Sample.mac
```
ターミナル上で起動して、コマンド入力待ち状態になる。

### バッチモードでの起動例

#### デフォルトでCUI環境で実行となる
```
$ ./bin/PTSdemo  -m  Sample.mac
```

#### スレッド数を指定して実行
スレッド数を12にする例
```
$ ./bin/PTSdemo -t 12  -m  Sample.mac
```
