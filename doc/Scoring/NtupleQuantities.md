# Score量の指定

Scoringで指定する識別名とその説明を示します。
基本的にスコアするセル単位で値を保存します。トラック番号が同じトラックのセル内でのエネルギー付与や線量値は、積算されて記録されます。また、その他の値については、セルに侵入した時点での情報、またはセル内で発生した時点での値を記録します。


| 識別名  |  説明 | 備考 |
|:--- | :----| :---- |
| evno|  イベント番号 | | 
| trkid | トラック識別番号  | |
| pid |   粒子種識別番号  | (注)PDGコード |
| de |  エネルギー付与値 |  セル内積算値 |
| dose |  線量値|   セル内積算値|
| stepL |  ステップ長 | セル内積算値  |
| stepNo | ステップ数 | セル内積算値  |
| ix |    セルの識別番号(1) |   |
| iy |     セルの識別番号(2) |   |
| iz |     セルの識別番号(3) |   |
| imod |   セルの識別番号(4) |   |
| isec |   セルの識別番号(5) |   |
| iext |   セルの識別番号(6) |   |
| w |     トラックの重み |   |
| proc |  初期粒子の反応を二次粒子以降が継承 | (注)プロセスタイプ  |
| ke |    運動エネルギー |   |
| x |     X座標 |   |

| riz |   RI生成のZ座標 | *試験運用  |
